# Seminaire Arrow

## Build

Pour builder lancer `quarto render` avec quarto.

Pour que le programme de compilation tourne il faut : 

- Quarto : https://quarto.org/docs/download/
- R : https://cran.r-project.org/bin/windows/base/
- Rtools : https://cran.r-project.org/bin/windows/Rtools/history.html
- les packages : arrow, lobstr, ggplot2, rmarkdown, tidyr, inline, nycflights13
- Conda/python installé et activé avec pyarrow et pandas dedans

## Publier

Pour publier sur github pages lancer `quarto publish` avec quarto et sélectionner github pages.
